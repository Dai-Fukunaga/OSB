# OSB
This is a repository for Operating System B

## Authors
Takatoshi Nakaguchi, Shusaku Nakajima, Dai Fukunaga

## How to use
### Compile
```bash
make
```

### Run Server A
```bash
java FileServer 8080
```

### Run Server B
```bash
java FileServer 8081
```

### Run Client
```bash
java Client <username>
```

### Clean exec files
```bash
make clean
```

### Clean temporary files
```bash
make fclean
```
